### How to use it on your local host.
~~~
git clone https://github.com/adityaazad79/IPL_Score_Predictor.git
~~~
~~~
cd IPL_Score_Predictor
~~~
~~~
pip install -r requirements.txt
~~~
~~~
streamlit run application.py
~~~
